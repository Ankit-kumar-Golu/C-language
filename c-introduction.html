C - Language

C is a procedural programming language. It was initially developed by Dennis Ritchie in the year 1972. It was mainly developed as a system programming language to write an operating system. The main features of the C language include low-level memory access, a simple set of keywords, and a clean style, these features make C language suitable for system programmings like an operating system or compiler development.

Element of C

  1. Character Set
    • Alphabets (Upper case, Lower case)
    • Digits 0-9
    • Special Character
  2. Word
    • Keyword
    • Identifier

1. Keyword - Reserved word - c has 32 keyword. keyword are always written in lower case.
Ex:- do - it is a keyword
Do - It is not a keyword

2. Identifier - Identifier are words that are used to identify any entity.

Rules for creating Identifier

1. It should consist of alphabets (upper and lower), digits and underscore only.

2. Identifier can't begin with digit.An identifier's 1st character must be alphabet or underscore.

3. A keyword can't be an identifier.

4. Because C is a case sensitive language so same word written in different case will be different identifier.

5. Length of identifier depends upon compiler.
